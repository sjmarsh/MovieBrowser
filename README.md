# MovieBrowser

Simple .Net Core & Vue.js app to browse and play movie files stored on a local drive.

### Instructions ###
1. Build web client:

    npm run build

2. Build/Package client and .net core app:

    dotnet-warp

3. Run the MovieBrowser.exe created in step 2
